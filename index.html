2018.12-2020.06
WXML
SCSS
JavaScript
Vue
Gulp

项目简介：

参与了「锦鲤多多」和「锦鲤内购」两个小程序项目以及后台管理系统的开发,其中「锦鲤多多」为活动小程序项目，「锦鲤内购」为 O2O 电商项目。
小程序项目使用「vue-on-wechat」框架，使得小程序拥有「Vue.js」一样的开发体验，支持：
  • 计算属性 computed 、监听属性 watch、Vuex 状态管理以及变量直接赋值。
  • 使用装饰器（Decorator）实现防抖（debounce）、节流 （throttle）、串行（series）。

后台管理项目采用「Ant Design of Vue」作为 UI 框架。

项目职责：

  • 使用微信小程序相关技术栈还原设计理念。
  • 负责项目中 UI 组件的封装以及 CSS3 动画效果的实现。

项目成果:

  • 通过对弹窗组件和动画的封装，让代码量减少 20%，并且为团队开发节省 15% 的开发时间。
  • 通过对「栅格布局」和 「Flex 布局」等布局代码的封装，让团队开发效率提升 10%。
  • 运用 Promise 异步等待特性对弹窗的弹出进行管理，让弹窗弹出变得井然有序。
